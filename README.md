# 🚀 AI-Powered Job Portal

An AI-powered full-stack job portal built to simplify job search and hiring with features like job recommendations, resume assistance, and chatbot support.

---

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Auth: Clerk
- AI: OpenAI API

---

## ✨ Features
- 🔐 Secure Authentication (Clerk)
- 👤 Role-Based Access (User / Recruiter / Admin)
- 💼 Job Posting & Management
- 📄 Apply & Track Applications
- 🤖 AI Job Recommendations
- 🧠 Resume Assistant
- 💬 AI Chatbot

---

## 🧠 AI Modules
- Job Recommendations → based on skills & profile  
- Resume Assistant → improve resume content  
- Chatbot → career help & guidance  

---

## 📌 Workflow

1. Authentication & Role Setup  
2. Job CRUD (Create, Read, Update, Delete)  
3. Application System  
4. Dashboard (User & Recruiter)  
5. AI Integration (OpenAI)  
6. Testing & Deployment
